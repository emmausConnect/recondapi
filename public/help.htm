Ce site peut être utilisé de 2 façons : * intéractivement * en tant qu'API *

INTERACTIF

  • connectez-vous au site
  • dans le champ "Demande", copiez les colonnes "cpu", "taille RAM" et "taille disque"
  • cliquez sur "calculer"
  • copiez/coller la réponse du champ "Résultat" dans votre Excel

API

Le mode API est prévu pour être utilisé par un programme, fonction Excel ...

dans l'URL, vous devez indiquer les infos suivantes :
/!\ ne pas oublié de faire l'encode d'URL (qui entre autre remplace les espaces par des "+" /!\

* query :
    cpu : seulement les info du CPU (indice et catégorie)
    pc  : évaluation du PC


* cpu[] : si on veut mettre plusieurs syntaxes pour le CPU
   le programme fera la recherche et renverra le résultat pour le premier qui fonctionne
    exemple : 
	si le CPU est connu sous 2 noms : 
	   AMD A10 PRO-7800B APU
	   et
	   AMD A10 PRO-7800B
	
	cpu[]=AMD+A10+PRO-7800B+APU&cpu[]=AMD+A10+PRO-7800B

* cpu : text du CPU (quand il n'y a qu'un seukl texte de CPU les [] sont inutiles) : default = ""


* tailleram  : taille de la ram sous forme "4g" "4 Go" ... : default = 0g
* taillehdd  :  taille de la ram sous forme "4g" "4 Go" ... : default = 0g
* outfmt     : format de sortie 
      json
	  text  :  dft = JSON
* separateur : utilisé si format "text". dft = "\t" (tabulation)
* detail     : 1 => le détail du calcul est joint au résultat (pour debug)
